# 郑商所白糖主力合约 EGARCH(1,1)-t 波动率建模

本项目基于郑商所白糖主力合约日收益率，按照传统 GARCH 类模型流程，完成从收益率检验、模型筛选、残差诊断，到波动率预测、VaR 风险边界和仓位缩放的完整波动率建模框架。

核心目标不是预测白糖涨跌方向，而是回答：

* 当前白糖处于高波动还是低波动状态；
* 下一交易日正常波动范围大概多宽；
* 多头 / 空头单日 VaR 风险边界在哪里；
* 当前仓位是否应因波动率变化而放大或缩小；
* 模型是否能较好刻画白糖收益率的厚尾和波动率聚集。

---

## 1. 建模逻辑

白糖主力合约样本量为 2536 个交易日。收益率均值接近 0，日收益率标准差约 0.84%，偏度接近 0，峰度约 3.87，高于正态分布的 3，说明收益率整体接近对称，但存在明显厚尾。

ADF 检验显示，对数价格序列不平稳，收益率序列显著平稳。因此，模型建立在日收益率 `ret_pct` 上，而不是价格或对数价格上。

Ljung-Box 与 ARCH-LM 检验显示，收益率本身线性自相关不强，但收益率平方在较长滞后下存在显著自相关，ARCH-LM 在 20 阶滞后下显著。这说明白糖收益率存在波动率聚集，即高波动和低波动会阶段性延续，具备使用 GARCH 类模型的基础。

---

## 2. 模型筛选结果

候选模型包括：

* ARCH(5)-t
* GARCH(1,1)-Normal
* GARCH(1,1)-t
* AR(1)-GARCH(1,1)-t
* GJR-GARCH(1,1)-t
* EGARCH(1,1)-t

所有候选模型均通过传统残差诊断，即标准化残差无明显自相关，标准化残差平方无明显自相关，ARCH-LM 未发现明显剩余 ARCH 效应。

在通过残差诊断的模型中，EGARCH(1,1)-t 的 AIC 和 BIC 最低：

| Model         |     AIC |     BIC |
| ------------- | ------: | ------: |
| EGARCH(1,1)-t | 5969.51 | 6004.54 |

因此，本文选取 EGARCH(1,1)-t 作为白糖主力合约收益率的主波动率模型。普通 GARCH(1,1)-t 和 GJR-GARCH(1,1)-t 可作为稳健性对照模型。GARCH(1,1)-Normal 虽然也通过残差诊断，但 AIC/BIC 明显劣于 t 分布版本，说明正态误差项不足以刻画白糖收益率的厚尾风险。

---

## 3. 最终模型解释

最终模型为：

> Constant Mean + EGARCH(1,1) + Student-t error

关键参数如下：

| Parameter  | Estimate | Interpretation               |
| ---------- | -------: | ---------------------------- |
| `mu`       | 0.000002 | 均值项接近 0，且不显著，说明模型不提供方向性预测    |
| `alpha[1]` | 0.057641 | 冲击大小项显著，说明大涨大跌会影响后续波动率       |
| `gamma[1]` | 0.019412 | 非对称项在 5% 水平下不显著，仅能说明存在弱非对称迹象 |
| `beta[1]`  | 0.989908 | 波动率持续性极强，高波动和低波动状态都会延续       |
| `nu`       | 4.497111 | t 分布自由度较低，说明收益率存在明显厚尾        |

模型最强的统计结论是：

1. 白糖收益率存在明显厚尾；
2. 白糖条件波动率具有强持续性；
3. 收益率冲击大小会显著影响未来波动率；
4. 非对称波动迹象存在，但在 5% 水平下证据不强。

## 5. VaR 风险边界与回测

基于 EGARCH(1,1)-t 和 Student-t 分布，下一交易日 VaR 结果如下：

| Risk Measure               | Return Threshold |
| -------------------------- | ---------------: |
| Long 95% VaR lower return  |         -1.0031% |
| Long 99% VaR lower return  |         -1.7131% |
| Short 95% VaR upper return |          1.0031% |
| Short 99% VaR upper return |          1.7131% |

按最新收盘价 5343 换算：

| Confidence             | Price Range |
| ---------------------- | ----------: |
| 95% one-day risk range | 约 5289–5397 |
| 99% one-day risk range | 约 5252–5435 |

该边界用于衡量统计意义下的单日风险范围，不代表价格一定触及，也不能直接等同于止损位。

Kupiec 无条件覆盖率检验显示，多头 95%、多头 99%、空头 95%、空头 99% VaR 均通过 5% 显著性水平检验：

| Side  | Confidence | Actual Exception Rate | Expected Rate | Decision |
| ----- | ---------: | --------------------: | ------------: | -------- |
| Long  |        95% |                 5.36% |         5.00% | Pass     |
| Long  |        99% |                0.946% |         1.00% | Pass     |
| Short |        95% |                 4.73% |         5.00% | Pass     |
| Short |        99% |                0.986% |         1.00% | Pass     |

这说明 EGARCH(1,1)-t 对白糖单日尾部风险的覆盖率整体合理，尤其 99% VaR 对极端尾部风险刻画较好。

## 6. 仓位缩放与交易参考

本文以历史中位数波动率 0.8323% 作为目标波动率。当前 EGARCH 条件波动率为 0.6445%，低于目标波动率，因此目标波动率法给出的截断后仓位缩放因子为：

| Item                      |   Value |
| ------------------------- | ------: |
| Target Volatility         | 0.8323% |
| Current EGARCH Volatility | 0.6445% |
| Position Scale            |  1.2914 |

这意味着：如果已有独立方向信号，当前低波动状态下可考虑将仓位调整至约 1.29 倍标准仓位。

但该因子只反映波动率风险预算，不提供方向信号。方向判断仍需来自趋势、基本面、价差结构、库存、进口、成交持仓或其他交易逻辑。

更准确地说：

* 方向信号决定做多还是做空；
* EGARCH 波动率决定仓位放大还是缩小；
* VaR 边界用于衡量单日尾部风险；
* 标准化残差用于监控异常冲击。

## 7. 实际价值与模型边界

EGARCH(1,1)-t 的最大价值，是把白糖从“单纯价格判断”转化为“动态风险状态判断”。

它可以用于：

* 识别低波动、正常波动、高波动和极高波动状态；
* 根据波动率动态调整仓位；
* 构造多头和空头 VaR 风险边界；
* 辅助止损宽度、保证金压力和单日风险预算判断；
* 识别实际收益率显著超出模型正常范围的异常冲击日；
* 作为交易系统中的风险过滤器，而非单独交易信号。

模型边界也需要明确：

EGARCH(1,1)-t 是波动率模型，不是方向预测模型。它不能判断白糖应该上涨还是下跌，也不能消除政策、天气、外盘原糖、产业消息和流动性冲击带来的跳跃风险。

当前模型的核心结论是：

> 白糖收益率存在明显厚尾，条件波动率具有强持续性；当前处于低波动压缩区；若存在独立方向信号，波动率目标法允许适度高于标准仓位；但是否交易、交易方向和具体止损仍需结合基本面、价格结构和交易纪律。

后续可进一步补充滚动样本外预测、Christoffersen VaR 回测、极端标准化残差日期复核，以及 Hong-Lee generalized spectral test 等高级稳健性检验。
