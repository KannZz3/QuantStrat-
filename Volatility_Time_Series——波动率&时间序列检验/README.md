# 商品期货多合约波动率建模与 Hong-Lee 谱充分性检验风控框架

本项目针对中国主流商品期货主力合约构建了一套严谨的波动率建模、统计健康度检验与动态风险管理系统。项目核心在于融合传统异方差诊断与 **Hong and Lee (2017) 广义谱波动率充分性检验**，解决传统线性检验对非线性、长滞后异方差残留的诊断盲区，并将其转化为交易执行端的动态风险预算（仓位缩放）与极值敞口控制（VaR）。

---

## 一、 项目定位与工作流

*   **项目定位**：非方向性价格预测模型，而是**资产风险状态识别与敞口管理风控模块**。
*   **核心工作流**：
    1.  **初级诊断**：日收益率平稳性（ADF）与线性自相关/异方差聚集（Ljung-Box, ARCH-LM）检验。
    2.  **候选模型池估计**：拟合包含 GARCH, EGARCH, TARCH, GJR-GARCH 的 19 种主流参数组合。
    3.  **BIC 最优筛选**：在通过标准化残差 Ljung-Box 和 ARCH-LM 传统异方差诊断的模型中，按 BIC 最小化规则确立最优模型。
    4.  **Hong-Lee 谱检验**：利用广义特征函数在频域构建谱密度，检验标准化残差是否仍余留高阶非线性或未知形式的广义波动率依赖。
    5.  **有限样本校准**：通过蒙特卡洛仿真（Monte Carlo Size/Power）评估谱检验在有限样本下的实际第一类错误（Size）。
    6.  **交易风控对接**：滚动预测 1d/5d/10d 波动率，通过 Kupiec 回测双向 VaR 边界，并结合模型健康度进行目标波动率仓位缩放（Volatility Targeting）。

## 二、 核心实证分析（基于 20 个主力合约 CSV 运行结果）

基于 20 个中国商品期货主力合约日频收益率的运行结果（详见数据文件 `01_` 至 `16_`），得出以下关键量化实证结论：

### 1. 最优模型选择：EGARCH 族与非正态厚尾分布的绝对统治
*   **非对称效应显著**：在 20 个合约中，**多达 15 个合约** 的最优模型被 BIC 选为 EGARCH 类（如 `EGARCH11_t`，`EGARCH21_t` 或 `AR1_EGARCH11_t`）。这印证了商品期货市场在面对利多与利空冲击时存在显著的波动率非对称（杠杆）效应。
*   **厚尾设定不可或缺**：所有品种的最优模型均排除了 Normal 分布，全部选择 Student-t、GED 或 Skewed Student-t 等厚尾分布。

### 2. 传统检验盲区穿透：Hong-Lee 谱检验的尖锐拒绝
*   **传统诊断的局限性**（见 `04_` 表）：所有 20 个选出的最优模型，其标准化残差的 Ljung-Box $z$ 和 $z^2$（10阶和20阶）以及 ARCH-LM 检验的 p 值均远远大于 0.05（全线 PASS），传统方法认为异方差已被完全解释。
*   **数据驱动谱检验穿透**（见 `13_` 表）：在引入**惩罚性数据驱动带宽选择器**后，Hong-Lee 检验以 5% 的显著性水平**显著拒绝了 3 个最优模型的充分性**：
    *   **AU (沪金)**：最优带宽 $p=5.32$，统计量 $M_{robust}=1.69$，p 值 = **0.0455**（拒绝）。
    *   **I (铁矿石)**：最优带宽 $p=5.32$，统计量 $M_{robust}=2.51$，p 值 = **0.0061**（强烈拒绝）。
    *   **P (棕榈油)**：最优带宽 $p=5.32$，统计量 $M_{robust}=1.85$，p 值 = **0.0324**（拒绝）。
*   **实证启示**：黄金、铁矿石和棕榈油即便通过了传统低阶检验，但在特定的短滞后带宽谱密度上，仍残留有高阶非线性或复杂的波动率自相关结构。

### 3. 模型健康度局限性：第四矩发散警报
*   **第四矩发散警报**（见 `10_model_health_summary.csv`）：**AG（沪银）**、**CF（棉花）** 等品种被标注为 `hl_inference_limited_fourth_moment`。
*   **原因与局限**：这些品种的 Student-t 自由度估计值 $\nu \le 4$（如 AG 仅为 3.1640），在数理上其第四矩（超额峰度）发散或不存在。这会导致基于第四矩的 Hong-Lee 渐进方差推导在数理上受限。

### 4. 蒙特卡洛 Size 仿真：极低的虚警率
*   **Size 校准结论**（见 `15_` 表）：在 5% 名义显著性水平下，以估计模型为 DGP 运行 100 次仿真，AL（GARCH-t）的实际拒绝率为 1%，AG（EGARCH-t）为 0%。这表明该统计量在有限样本下极度保守，在实盘中**能有效避免模型失效的虚警（False Alarms）**。

## 三、 实际交易与风险管理意义

本框架将统计检验直接映射为执行端的风控动作，具有极强的实盘指导价值：

```
                           +------------------------+
                           | 独立的多空交易方向信号 |
                           +-----------+------------+
                                       |
                                       v
+------------------------+  +----------+----------+  +------------------------+
|   EGARCH 条件波动率    |  |                     |  |   Hong-Lee 模型健康度  |
| 评估当前处于何种波动区间|  |  目标波动率仓位缩放  |  |    评估风险输出置信度  |
| (如 low_vol / high_vol)+->|   (Position Scale)  |<-+ (健康/不健康/第四矩受限)|
+------------------------+  +----------+----------+  +------------------------+
                                       |
                                       v
                           +-----------+------------+
                           |    最终执行交易仓位    |
                           +------------------------+
```

1.  **动态风险预算（Volatility Targeting）**：
    依据最新条件波动率 $\sigma_t$ 动态调节仓位（$\text{Raw Scale} = \sigma_{target} / \sigma_t$）。在波动率极度压缩时释放敞口，在波动率飙升时收缩敞口。
    *   *实盘案例*（2026年6月18日）：**AG（沪银）** 波动率暴涨至历史 95.86% 分位数，条件波动率为 3.805%，仓位自动收缩至原标的的 **0.417x**；而正常波动率下的 **AL（沪铝）** 释放仓位至 **1.091x**。
2.  **统计防御降权（Health Multiplier）**：
    对于被 Hong-Lee 谱检验拒绝（AU, I, P）或因第四矩发散而受限（AG, CF）的品种，其 EGARCH 模型对尾部风险存在潜在的低估。系统会自动触发降权乘数（健康乘数打折），调低其风险预算上限，提供二级防御。
3.  **极值敞口预警（VaR 滚动回测）**：
    利用通过 Kupiec 检验的最优模型生成日度多空 VaR。例如，白糖在收盘价 5343 下，99% 置信度的单日多空安全价格边界为 [5252, 5435]，为实盘期现套保或杠杆交易提供精确的止损与爆仓防御垫。
