# 商品期货多合约波动率建模与 Hong-Lee 谱充分性检验风控框架

本项目针对中国主流商品期货主力合约构建了一套波动率建模、统计健康度检验与风险管理参考系统。项目核心在于融合传统异方差诊断与 **Hong and Lee (2017) 广义谱波动率充分性检验**，评估传统线性检验对非线性、长滞后异方差残留的诊断效果，并尝试将其转化为交易风险管理中的动态风险预算（仓位缩放）与极值敞口控制（VaR）。

---

## 一、 项目定位与工作流

*   **项目定位**：非方向性价格预测模型，而是**资产风险状态评估与敞口管理风控参考模块**。
*   **核心工作流**：
    1.  **初级诊断**：日收益率平稳性（ADF）与线性自相关/异方差聚集（Ljung-Box, ARCH-LM）检验。
    2.  **候选模型池估计**：拟合包含 GARCH, EGARCH, TARCH, GJR-GARCH 的 19 种参数组合。
    3.  **BIC 模型筛选**：在通过标准化残差 Ljung-Box 和 ARCH-LM 传统异方差诊断的模型中，依据 BIC 最小化规则筛选出最优模型。
    4.  **Hong-Lee 谱检验**：利用广义特征函数在频域构建谱密度，检验标准化残差是否仍余留高阶非线性或特定形式的广义波动率依赖。
    5.  **有限样本校准**：通过蒙特卡洛仿真（Monte Carlo Size/Power）评估谱检验在有限样本下的经验第一类错误（Size）。
    6.  **风险控制参考**：滚动预测 1d/5d/10d 波动率，通过 Kupiec 回测评估双向 VaR 边界的覆盖效果，并结合模型健康度进行目标波动率仓位缩放（Volatility Targeting）参考。

## 二、 核心实证分析（基于 20 个主力合约 CSV 运行结果）

基于 20 个中国商品期货主力合约日频收益率的运行结果（详见数据文件 `01_` 至 `16_`），得出以下关键量化实证结论：

### 1. 最优模型选择：EGARCH 族与非正态厚尾分布占多数
*   **非对称效应表现**：在 20 个被测合约中，有 **15 个合约** 的最优模型被 BIC 选为 EGARCH 类（如 `EGARCH11_t`，`EGARCH21_t` 或 `AR1_EGARCH11_t`）。这表明在所选样本区间内，商品期货市场中较多品种表现出波动率非对称（杠杆）效应。
*   **厚尾设定适用性**：所有品种的最优模型均排除了 Normal 分布，全部选择 Student-t、GED 或 Skewed Student-t 等厚尾分布，这与商品期货日收益率普遍具有厚尾特征的数据属性一致。

### 2. 异方差检验差异：部分品种被 Hong-Lee 谱检验拒绝
*   **传统诊断表现**（见 `04_` 表）：在设定的滞后阶数下，所有 20 个选出的最优模型，其标准化残差的 Ljung-Box $z$ 和 $z^2$（10阶和20阶）以及 ARCH-LM 检验的 p 值均大于 0.05，未能拒绝无自相关及无剩余异方差的原假设。
*   **数据驱动谱检验表现**（见 `13_` 表）：在引入**惩罚性数据驱动带宽选择器**后，Hong-Lee 检验在 5% 的显著性水平下**拒绝了 3 个最优模型的充分性原假设**：
    *   **AU (沪金)**：最优带宽 $p=5.32$，统计量 $M_{robust}=1.69$，p 值 = **0.0455**（拒绝原假设）。
    *   **I (铁矿石)**：最优带宽 $p=5.32$，统计量 $M_{robust}=2.51$，p 值 = **0.0061**（拒绝原假设）。
    *   **P (棕榈油)**：最优带宽 $p=5.32$，统计量 $M_{robust}=1.85$，p 值 = **0.0324**（拒绝原假设）。
*   **实证启示**：黄金、铁矿石和棕榈油即便在低阶传统检验下未表现出显著异常，但在特定的数据驱动带宽谱密度下，表现出统计显著的非线性或特定滞后阶数的波动率自相关残留迹象。

### 3. 模型健康度提示：部分品种第四矩受限
*   **矩条件警报**（见 `10_model_health_summary.csv`）：**AG（沪银）**、**CF（棉花）** 等品种被标注为 `hl_inference_limited_fourth_moment`。
*   **原因与局限**：这些品种的 Student-t 自由度估计值 $\nu \le 4$（如 AG 仅为 3.1640）。在数理上，当自由度 $\nu \le 4$ 时，第四矩（峰度）发散或不存在，这可能使得基于第四矩的 Hong-Lee 谱检验渐进方差推导在数理上受限。

### 4. 蒙特卡洛 Size 仿真：表现出较低的经验拒绝率
*   **Size 校准结论**（见 `15_` 表）：在 5% 名义显著性水平下，以估计模型为 DGP 运行 100 次仿真，AL（GARCH-t）的经验第一类错误拒绝率为 1%，AG（EGARCH-t）为 0%。这表明该统计量在有限样本下表现倾向于保守，在实际应用中可能有助于降低模型失效的虚警概率。

## 三、 风险管理参考意义

本框架尝试将统计检验信息映射为动态仓位调节的参考信号：

```
                           +------------------------+
                           | 独立的多空交易方向信号 |
                           +-----------+------------+
                                       |
                                       v
+------------------------+  +----------+----------+  +------------------------+
|   EGARCH 条件波动率    |  |                     |  |   Hong-Lee 模型健康度  |
| 评估当前条件波动率区间  |  |  目标波动率仓位缩放  |  |    评估风险输出置信度  |
| (如 low_vol / high_vol)+->| (Position Scale参考)|<-+ (健康/不健康/第四矩受限)|
+------------------------+  +----------+----------+  +------------------------+
                                       |
                                       v
                           +-----------+------------+
                           |    风控参考仓位乘数    |
                           +------------------------+
```

1.  **动态风险预算（Volatility Targeting）**：
    依据条件波动率预测值 $\sigma_t$ 调整仓位参考值（$\text{Raw Scale} = \sigma_{target} / \sigma_t$），旨在平抑波动率变化对资产组合的风险贡献。
    *   *数据实操案例*（2026年6月18日）：**AG（沪银）** 波动率处于历史 95.86% 分位数，条件波动率为 3.805%，对应的仓位缩放参考系数收缩至 **0.417**；而正常波动率下的 **AL（沪铝）** 仓位缩放参考系数为 **1.091**。
2.  **模型健康度修正（Health Multiplier）**：
    对于被 Hong-Lee 谱检验拒绝（AU, I, P）或因第四矩发散而受限（AG, CF）的品种，由于模型对其尾部风险可能存在潜在低估，在风控框架中，可相应引入降权乘数（健康乘数）调低其仓位参考上限。
3.  **极值敞口参考（VaR 滚动回测）**：
    利用通过 Kupiec 检验的最优模型生成日度多空 VaR，为套期保值或杠杆控制提供量化的价格参考区间。例如，白糖在收盘价 5343 下，99% 置信度的单日多空安全价格边界为 [5252, 5435]。
