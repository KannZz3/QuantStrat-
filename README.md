# QuantStrat 量化金融研究项目复现与拓展平台 (UIUC MSFE 2026)

本项目是 **QuantStrat** 与**伊利诺伊大学金融工程硕士项目（UIUC MSFE）** 2026年夏季的量化金融研究合作项目。项目严格采用科学研究范式（Scientific Method），以文献理论为锚，针对加密货币与中国商品期货主力合约，完成了一套包含**基本面锚定、行为金融异象、高阶异方差频域检验以及动态风险控制**的完整量化研究与执行参考系统。

目前项目已全部完成三大支柱研究模块的开发与实证分析：

## 一、 已完成核心模块全景与成果

### 1. Crypto_Pricing —— 加密货币统一多维定价与下沿估值
*   **主要目的**：针对无贴现现金流特性的加密资产（BTC），探索其链上基本面的底层公允价值与市场行为折价的动态变化，确立合理的下沿估值区间。
*   **核心逻辑**：级联融合三篇顶尖学术论文：
    1.  **BDK 链上基本面价值锚 (Bhambhwani et al., 2019)**：以算力与活跃地址数作为自变量，计算算力下行与用户规模萎缩情景下的底层安全价值。
    2.  **Biais 均衡交易折价层 (Biais et al., 2023)**：加权评估交易收益（40%）、交易成本（20%）、市场准入（20%）与崩盘风险（20%），映射折价系数。
    3.  **Liu-Tsyvinski 风险注意力折价层 (Liu & Tsyvinski, 2021)**：结合价格动量（40%）、普通关注度（25%）、负面关注度（20%）与活跃度增长率（15%），测度共识溢价。
*   **主要成果与设计亮点**：
    *   构建了基于 Python 的学术级数据交叉验证系统 (`validator.py`)，包含 lead/lag 延迟相关性检验（Spearman Correlation）以及注意力指标的双源检验。
    *   设计了动态模型降级（Full Model $\rightarrow$ Core $\rightarrow$ BDK Only）与样本天数不足惩罚机制，确保估值区间（Price Range）的宽度能真实反映底层数据置信度。
    *   严格规避重复计数偏误，网络规模变量（Active Addresses）仅被允许在基本面价值锚中使用，在行为折价模块中被禁止二次计算。

### 2. Market_Anomalies —— 商品期货技术分析异象、活跃度衰减与发表后衰减
*   **主要目的**：检验“高波动资产中技术异象更强”的横截面收益假说，测试高交易活跃度状态是否会削弱该异象，以及量化策略在“样本内筛选-发表/选择后”的样本外（OOS）与伪实盘（Pseudo-Live）阶段的衰减效应。
*   **实证成果（基于 27 个主力商品期货日频 CSV 运行结果）**：
    *   **高波动技术异象局部且不显著**（Han et al. 2013复刻）：在 `High-minus-Low` 波动率桶的测试中，8个技术指标中有5个年化超额收益为正（如 MA20 年化 5.27%），但在 5% 显著性水平下 t 统计量全线不显著（最高仅为 0.96）。长周期指标（如 MA120）甚至表现出反转（年化 -10.23%，t = -1.77）。
    *   **活跃度对异象呈现方向性弱衰减**（Chordia et al. 2014复刻）：以成交量、持仓量、 turnover 合成的交易活跃度 z-score 显示，24 个测试中 18 个表现出衰减方向（如 MA120 HML 收益在低活跃度下为 1.20%，高活跃度下为 -16.44%，相差 -22.24%），但统计显著性不强（t = -1.47）。
    *   **选择后衰减是最强、最稳定的结果**（McLean & Pontiff 2016复刻）：在 IS 阶段（2016-2020）筛选出的 Top 10 策略，在 OOS（2021-2023）与 Pseudo-Live（2024-2026）阶段发生系统性滑坡：
        
        | 阶段 (Split) | 平均年化收益率 | 平均夏普比率 | 夏普保留率 (Sharpe Retention) | 衰减特征 |
        | :--- | :---: | :---: | :---: | :--- |
        | **IS (样本内)** | 5.96% | 0.68 | 100.0% | 基准筛选 |
        | **OOS (样本外)** | 3.72% | 0.35 | 55.5% | 收益中位保留率约 66.6% |
        | **PSEUDO_LIVE (伪实盘)**| 1.06% | 0.11 | 16.6% | 大幅塌陷，2个策略彻底转负 |

### 3. Volatility_Time_Series —— 波动率建模与 Hong-Lee 谱充分性检验
*   **主要目的**：引入频域谱分析，对经过传统诊断（Ljung-Box, ARCH-LM）合格后的最优 GARCH 波动率模型，进行更深层次的标准化残差“充分性检验”，发掘残留的高阶矩依赖与非线性异方差结构，并将其转化为动态风控参考。
*   **实证成果（基于 20 个主力商品期货日频 CSV 运行结果）**：
    *   **EGARCH-t 模型占多数**：20 个被测合约中，有 15 个被 BIC 筛选为非对称 EGARCH 族。所有 20 个品种的最优模型均采用非正态厚尾假定（Student-t/GED），印证了商品收益率的厚尾分布特征。
    *   **谱检验穿透传统盲区**（Hong & Lee 2017复刻）：最优模型在标准化残差的传统诊断中全线 PASS（p > 0.05）。但在数据驱动最佳带宽检验下，**AU (沪金)**（p = 0.0455）、**I (铁矿石)**（p = 0.0061）、**P (棕榈油)**（p = 0.0324）三品种的充分性原假设被显著拒绝，暴露出传统异方差诊断的盲区。
    *   **实战动态风控仓位调节**：
        *   **动态风险预算（Volatility Targeting）**：根据条件波动率调整风险敞口。如在 2026年6月18日，因银价大涨，**AG（沪银）** 条件波动率高达 3.805%（历史 95.86% 分位数），仓位因子自动收缩至 **0.417x**；而平稳波动下的 **AL（沪铝）** 仓位为 **1.091x**。
        *   **健康度防御修正**：对于被谱检验拒绝（AU, I, P）或第四矩发散的品种（AG, CF），引入健康降权因子调低其仓位参考上限，规避潜在的尾部风险低估。

## 二、 量化研究规范与工程设计亮点

1.  **时序逻辑严密（无 Look-ahead Bias）**：
    所有策略回测均执行延迟交易原则。信号在 `t-1` 收盘后形成，仓位在 `t` 日开盘或收盘执行，收益窗口为 `close[t] -> close[t+1]`。样本切分使用完整 Return Window 以杜绝跨区间收益渗透。
2.  **统计工具的稳健性**：
    *   横截面回归采用 OLS 结合 **HC1 异方差稳健标准误** 评估显著性。
    *   蒙特卡洛有限样本 Size 校验（仿真 100 次）采用 **Wilson Score 置信区间** 代替 Wald 渐进区间，防止边界概率塌陷为零。
3.  **高性能数据管道**：
    系统数据流全部基于 Parquet 高速读写，数据源涵盖 CoinGecko, Binance, Yahoo Finance, Farside, GDELT 等十余个接口，支持离线数据资产在 Git 上的完全可复现性。
